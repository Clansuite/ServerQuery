Clansuite Server Query

Clansuite Server Query currently supports 33 server protocols and is compatible with 146 game and voice servers.

This document provides a complete list of supported protocols and games.

If the server you are looking for is not included, you can either contribute by forking the project on GitHub and add it yourself, or submit a request to have it added.

Best regards,
Jens A. Koch
